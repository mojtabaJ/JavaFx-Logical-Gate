## Logical Gate Simulator
Logical Gate JavaFx

* This is a application that helps the circuit designer to test the working of their circuit.

## Logical Gates 
1. AND.
2. OR.
3. NAND.
4. NOR.
5. XOR.
6. XNOR.
7. NOT.

## Features
1. Circuit Testing.
2. Simple Drag and Drop interface.
